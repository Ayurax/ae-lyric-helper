## ğŸ¬ What It Does

Paste your full lyrics into the panel.

Then:
- Click **Next Line** to apply lines sequentially to a selected text layer.
- Click **Next + New Text** to create new timed text layers automatically.
- Use **Duration (sec)** to control layer length.
- Select multiple text layers to fill them in order.

Line numbers are displayed in the panel for easy navigation, but are automatically removed when applied to layers.
